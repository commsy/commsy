Notes:
This is a major update of CommSy. The layout went through a major redesign, 
some funtionality was changed and the usabilty was further improved.

- improved design of list views
- new form handling
- new configuration pages
- copy and paste of entries between rooms
- private room functionality
- import of dates via csv
- extension: new room website
- new search feature for each room
