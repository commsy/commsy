CommSy Version 6.0.0

Features:
-----
Folgende Neuerungen stecken in CommSy 6.0.0

    * Bildvorschau
    * Einbetten von Audiodateien
    * Einbetten von Bildern
    * Einbetten von Dateien
    * Einbetten von Medien und anderen Dateitypen
    * Einbettung von Filmen
    * Installationsroutine
    * Kategorien in allen Rubriken
    * Löschen auf Detail- und Listen-Ansichten
    * Mathematische Formeln
    * Mitteilungsbox
    * Mootools
    * Neue englische Sprachfassung
    * Neuer Chat
    * Out-of-Service-Schalter
    * Raumvorlagen als "Standard" vorgeben
    * Sprachumschalter
    * Verbessertes Design
    * Verlinkung (neue Syntax)
Detailliertere Informationen: http://www.commsy.net
