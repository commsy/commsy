CommSy 5.1.0 (PCommSy build 574)

This is a interim release update of CommSy. Know bugs have been addressed and
some features have been added.

One major feature added is an enhancement for file uploads. We now support a client side 
plugin that enables you to upload multiple files and at the same time get progress indication.
