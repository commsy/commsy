CommSy 5.2.0 (PCommSy build 680)

This is a interim release update of CommSy. Know bugs have been addressed and
some features have been added.

One major feature added is an enhancement for file uploads. 
We now support uploading a file directly during creating 
an entry. Consequently, the searchbox located at the home page
can find files with these attachments. Also, this release introduces 
threaded discussions. On the portal settings side, the behaviour of the
portal can be configured regarding the connection between project and
community rooms.
