Vorgehensweise:
Wenn man eine Rubrik umbenennbar machen möchte, muss man folgendes beachten:
1. Im Extrafeld des Raumes, dessen Rubrik man umbennen möchte, muss ein Array der oben angegebenen Form eingetragen werden. Hierbei entspricht:

S = Singular
P = Plural
1 = Nominativ
2 = Genitiv
3 = Akkusativ
4 = Dativ
GENUS = Geschlecht: Zur Auswahl stehen N = Neutrum, F = Feminin, M = Maskulin

2. Es muss ein Datanbankskript geben, dass bei einem Update den Standard-Array mit den Originalbegriffen in alle betreffenden Raum-Items einträgt.

3. Im Initialisierungsprozess von Räumen müssen initiale Werte in die Extras-Felder geschrieben werden.

4. In den entsprechenden Konfigurationsbereichen muss eine Formular enstehen, mit dem die entsprechenden Werte gesetzt werden können.
